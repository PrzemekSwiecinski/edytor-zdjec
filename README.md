# Edytor zdjęć w Pythonie
(PL) Edytor zdjęć w języku Python z 16 funkcjami edycji pojedynczego zdjęcia, 16 funkcjami mieszania obrazów oraz histogramem. Menu stworzone przy pomocy biblioteki tkinter. Program zawiera również możliwość cofnięcia każdej zmiany oraz zapisu aktualnego obrazu jako "aktualny_obraz.jpg". Histogram również może być uruchomiony dla każdego etapu edycji zdjęcia.

(EN) A Python photo editor with 16 single-photo editing functions, 16 image blending functions, and a histogram. The menu is created using the tkinter library. The program also includes the ability to undo any changes and save the current image as "current_image.jpg." The histogram can also be activated for each stage of the photo editing process.
